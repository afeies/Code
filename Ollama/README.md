1. list models
- ollama list

2. remove model
ollama rm <model-name>

3. add download model
- go to https://ollama.com and search models
- ollama run <model-name>
    - pulls and runs in terminal
    - same thing as ollama app

## Models
- LLava:7b: https://ollama.com/library/llava:7b
    - TennisVision AI: similar OpenAI API, supports images, small enough to run locally
